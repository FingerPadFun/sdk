# FingerPad

**FingerPad** is a secure, browser-compiled note environment designed to protect your thoughts.
Everything is encrypted with your own keys before it ever leaves your device.

No servers. No tracking. Just you and your thoughts.

## 🧠 About FingerPad

FingerPad is built for people who value privacy.
Your notes are compiled directly in your browser and encrypted locally using your personal keys.

That means:

* Your data never leaves your control
* No centralized storage risks
* True client-side security

FingerPad is the only place where your thoughts stay truly yours.

---

## 🔐 Key Features

* **Client-side Compilation**
  FingerPad runs entirely in your browser.

* **End-to-End Encryption**
  All notes are encrypted with keys generated by you.

* **Zero Data Storage**
  No centralized servers storing your thoughts.

* **Private by Design**
  No tracking, no analytics, no surveillance.

* **Instant Access**
  Start writing securely in seconds.

---

## ⚙️ How It Works

1. Open FingerPad in your browser.
2. Generate your encryption key.
3. Start writing your notes.
4. Everything is encrypted locally before storage.

Your keys. Your notes. Your privacy.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/fingerpad.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
